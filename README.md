# 🏨 Hotel Management System (Java + Oracle)

## 📌 Description
This project is a Java-based Hotel Management System developed during my BCA to manage hotel operations like room booking, customer records, and report generation.

## 🚀 Features
- User Login System
- Room Booking & Management
- Customer Record Management
- Report Generation

## 🛠️ Technologies Used
- Java (Swing, AWT)
- Oracle Database (RDBMS)
